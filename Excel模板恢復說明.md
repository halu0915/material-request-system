# Excel 模板恢復說明

## 📋 需求

恢復之前的 Excel 模板，包含三個分頁：

1. **叫料單明細** - 當前叫料單的材料明細
2. **月統計** - 累加當月歷史所有的材料統計
3. **圖片/連結** - 如果建立叫料單時有上傳圖片或連結，自動在第三個分頁產生

## 🔧 需要實現的功能

### 1. 數據庫修改

需要在 `material_requests` 表中添加欄位來存儲圖片/連結：
- `images` (TEXT[]) - 存儲圖片 URL 或路徑
- `links` (TEXT[]) - 存儲連結 URL

### 2. Excel 生成功能

修改 `generateExcel` 函數以支持三個分頁：

#### 分頁 1: 叫料單明細
- 顯示當前叫料單的所有材料項目
- 包含：材料類別、材料名稱、數量、單位、備註

#### 分頁 2: 月統計
- 查詢當月（與當前叫料單同一個月）的所有歷史叫料單
- 按材料類別和材料名稱分組統計
- 累加每種材料的總數量

#### 分頁 3: 圖片/連結
- 如果有上傳圖片，顯示圖片連結或嵌入
- 如果有上傳連結，顯示連結列表
- 如果沒有圖片或連結，此分頁可以為空或顯示提示

### 3. 文件上傳功能

需要添加文件上傳 API 來處理圖片上傳。
