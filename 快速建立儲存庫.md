# ⚡ 快速建立 GitHub 儲存庫

## 🎯 最簡單的方法（3 步驟）

### 步驟 1: 在 GitHub 建立儲存庫

直接點擊這個連結（會自動開啟建立頁面）：
👉 **https://github.com/new**

或手動前往：
1. 登入 https://github.com
2. 點選右上角「+」→「New repository」件
3. 輸入：
   - Repository name: `material-request-system`
   - Description: `叫料系統 - Material Request System`
   - 選擇 Private 或 Public
   - **⚠️ 不要勾選任何初始化選項**
4. 點選「Create repository」

### 步驟 2: 複製儲存庫 URL

建立完成後，GitHub 會顯示您的儲存庫 URL，類似：915
```
https://github.com/您的帳號/material-request-system.git
```

複製這個 URL。

### 步驟 3: 推送程式碼

在終端執行：

```bash
cd "/Users/halu_1/cursorAI/叫料系統"
git remote add origin https://github.com/您的帳號/material-request-system.git
git push -u origin main
```

**或使用自動化腳本：**

```bash
./push-to-github.sh
```

腳本會詢問您儲存庫 URL，貼上即可。
